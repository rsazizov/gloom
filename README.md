# Gloom

Gloom is an OpenGL based DOOM renderer.

## Requirements

* glm
* glfw3
* glew

All the requirements can be installed with the following command (on Ubuntu/Debian)

```
sudo apt install libglm-dev libglfw3-dev libglew-dev
```

## Usage

```
mkdir bin obj
make
./bin/gloom <file.wad>
```
